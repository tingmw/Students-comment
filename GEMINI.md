# 小學生學期末評語產生器
## 目的
- 透過複選網頁上的學生特質，讓老師利用該網頁為國小學生生成相對應的期末評語

## 開發技術
- flask
- HTML, CSS, JavaScripts
- Gemini LLM API

## 功能
### 核心功能
- 網頁上呈現下面提供的學生特質清單讓用戶選取
- 網站藉由選取的該學生特質，透過 LLM 的 API 調用 LLM 生成客製化的學生評語

### 產生評語的回應風格
- 正向積極的回應，不要負面風格
- 字數約 100 出頭字

### 附加功能
- 網頁提供一個輸入框可以輸入學生姓名，讓產生的評語回應由這位學生的名字作為主詞

## 頁面
- 分成左右側頁面，左側佔據整個頁面的 35% 比例，右側佔據 65%

### 左側頁面
- 最上面是一個輸入框，讓用戶可以輸入學生姓名
- 中間提供下面的學生特質清單，並且以一個個標籤的形式呈現，然後可以複選
- 下面為一個按鈕，當用戶選擇完畢可以透過此按鈕提交學生的特質

### 右側頁面
- 輸出 LLM 生成的回應，並且於呈現框右上角提供一個複製按鈕讓用戶可以複製回應
